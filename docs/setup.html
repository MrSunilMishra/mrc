Adding a New Page

  1. Create a new .html file (copy index.html or docs/setup.html as a template).
  2. Ensure the <link rel="stylesheet" href="..."> points correctly to css/style.css.
  3. Ensure the <script src="..."> points correctly to js/main.js.
  4. Add your content inside the <main> tag.
  5. Add the fade-in class to sections you want to animate on scroll.
  6. Update the navigation menu in all pages (or move the header to a JS component if you know JS).
