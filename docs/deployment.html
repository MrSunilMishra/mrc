Hosting Your Site on GitHub Pages

GitHub Pages is a free hosting service for static sites directly from your GitHub repository.

1. Create a Repository

  1. Log in to GitHub.
  2. Click New to create a repository.
  3. Name it (e.g., my-pyscript-site) and ensure it is Public (for free Pages).

2. Push Your Code

Open your terminal in the project folder and run:

# Initialize git
git init

# Add all files
git add .

# Commit
git commit -m "Initial commit"

# Link to your repo (replace URL)
git remote add origin https://github.com/Start-Code/my-pyscript-site.git

# Push
git push -u origin main

3. Configure Pages

  1. Go to your repository Settings tab.
  2. Click on Pages in the left sidebar.
  3. Under Branch, select main and the folder / (root).
  4. Click Save.

Wait a minute or two, refresh the page, and you will see your live URL (e.g., https://username.github.io/my-pyscript-site/).

4. PyScript Considerations

Note: PyScript fetches the Python runtime (WASM) from the web. The first load might be slightly slower. Ensure your users have an internet connection.
