Getting Started with ESP32 & MicroPython

A complete step-by-step guide to setting up your environment and running Python on hardware.

1. Install Thonny IDE

Thonny is the most beginner-friendly Python IDE with built-in support for MicroPython.

  1. Download Thonny from thonny.org.
  2. Install it on your computer (Windows, Mac, or Linux).
  3. Open Thonny.

Why Thonny? It handles drivers and serial connections automatically for most boards.

2. Flash MicroPython Firmware

Your ESP32 needs the MicroPython firmware to understand Python code.

Steps to Flash:

  1. Connect your ESP32 to your PC via USB data cable.
  2. In Thonny, go to Tools > Options > Interpreter.
  3. Select MicroPython (ESP32) from the dropdown.
  4. Click Install or Update MicroPython (bottom right).
  5. Select the target port and firmware variant (usually default is fine).
  6. Click Install and wait for it to finish.

3. Your First Code: Blink LED

The "Hello World" of electronics. Most ESP32 boards have a built-in LED on GPIO 2.

from machine import Pin
import time

# Setup built-in LED (GPIO 2)
led = Pin(2, Pin.OUT)

while True:
    led.value(1)    # Turn ON
    time.sleep(0.5) # Wait 0.5 seconds
    led.value(0)    # Turn OFF
    time.sleep(0.5)

To Run: Paste this into Thonny and click the green "Run" button.

4. Connecting to Wi-Fi

ESP32 is powerful because of its Wi-Fi capabilities. Here is a helper function to connect.

import network
import time

def connect_wifi(ssid, password):
    wlan = network.WLAN(network.STA_IF)
    wlan.active(True)
    if not wlan.isconnected():
        print('Connecting to network...')
        wlan.connect(ssid, password)
        while not wlan.isconnected():
            pass
    print('Network config:', wlan.ifconfig())

# Usage
connect_wifi("MyWiFiName", "MyPassword")

5. Using Sensors (DHT11/22)

Reading temperature and humidity is easy with the built-in dht module.

import dht
from machine import Pin
import time

sensor = dht.DHT22(Pin(14)) # Connect sensor data pin to GPIO 14

while True:
    try:
        sensor.measure()
        temp = sensor.temperature()
        hum = sensor.humidity()
        print(f"Temperature: {temp}Â°C, Humidity: {hum}%")
    except OSError as e:
        print("Failed to read sensor.")
    
    time.sleep(2)

Troubleshooting

  • Port not found? Install the CP210x or CH340 drivers depending on your board.
  • Brownout reset? Try a different USB cable or USB port (power issue).
  • Thonny busy? Press the 'Stop' (red sign) button to interrupt the running script.
