Python Meets HTML

With PyScript, you can interleave Python logic with HTML. The example on the right shows how simple it is to print "Hello World" or perform calculations directly in the browser. 

<script type="py">
    import datetime
    now = datetime.datetime.now()
    print(f"Current time: {now}")
</script>
